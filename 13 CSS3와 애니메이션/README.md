#  변형 

## 2차원 변형과 3차원 변형 

- <code>2차원 변형(2D Transform)</code> 
  - 웹 요소를 변형시킬 때 단순히 수평이나 수직으로 이동하고 회전하는 것을 말합니다.
  - 수평이나 수직으로 옮기는 것은 x축과 y축으로 이동하는 것이기 때문에 2차원 좌표를 사용힙니다. 

- <code>3차원 변형(3D Transform)</code>
    - x축과 y축에 원근감을 주는 z축을 추가해 변형시키는 것을 말합니다.
